# Smart Brain - Fullstack React App

## Overview of Project
- Smart Brain is a fullstack web app that uses the Clarifai API to locate a face in a (PNG/JPG) image link.
- Features include: 
  - Profile Registration Form
  - Sign-In/Sign-Out
  - Face Detection API (Clarifai)
  - Total Entry Count
